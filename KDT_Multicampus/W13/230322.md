## Django - Django Template

## Template System

- django template system
    - 데이터 표현을 제어하면서, 표현과 관련된 로직을 담당
- HTML 컨텐츠 => 변수 값에 따라 변화

  ```python
  from django.shortcuts import render
  def index(request):
      context = {
          'name': 'Harry',
      }
      return render(request, 'articles/index.html', context)
  ```

  ```html
  <body>
    <h1>Hello, {{ name }}!!</h1>
  </body>
  ```
